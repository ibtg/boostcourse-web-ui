- 프로젝트는 다음과 같은 사항을 고려하였습니다.

  - 시맨틱 마크업

  - 크로스 브라우징

  - HTML 코드 유효성 검사 ([https://validator.w3.org/](https://validator.w3.org/))

  - CSS 코드 유효성 검사 ([https://jigsaw.w3.org/css-validator/](https://jigsaw.w3.org/css-validator/))
